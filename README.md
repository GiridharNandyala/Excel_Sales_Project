# Excel_Sales_Data_Analysis

Description:
This project focuses on analyzing retail sales data using Microsoft Excel to extract meaningful insights and support data-driven decision-making.

Key Features:

Cleaned and structured raw sales data
Created interactive dashboards using Pivot Tables & Charts
Analyzed sales performance by product, region, and time
Identified top-selling products and revenue trends
Used formulas like VLOOKUP, IF, SUMIF for data processing

Tools & Technologies:

Microsoft Excel
Pivot Tables
Charts & Dashboards
Data Cleaning Techniques

Insights:

Identified high-revenue products and peak sales periods
Compared regional sales performance
Detected patterns to improve business strategies

Outcome:

Built a dynamic dashboard for quick business insights
Improved data interpretation and reporting skills
